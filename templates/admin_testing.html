Automated Network Tests

One-click automated tests are available on the ESP32 admin dashboard:

  1. Quick Health Check — Verifies XBee TX/RX counters and link assessment
  2. Network Performance — Sends N packets, measures packet loss % and throughput (pass: <2% loss)
  3. One-Way Delay — Measures round-trip time at 5 payload sizes (1B, 10B, 30B, 50B, 70B)
  4. Payload Size & Load — Tests message rate at different payload sizes vs programmed rate
Connect to HopFog-Network WiFi and open http://192.168.4.1/admin/messaging/testing to run automated tests.
